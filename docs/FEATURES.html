üí∞ Rate Management

Flexible pricing system supporting base rates and special promotional pricing.

Base Rates

  • Set default rates for 1 or 2 guests
  • Only one active base rate per guest count
  • Automatically displayed on public website
  • Default: R850 (1 guest), R950 (2 guests)

Special Rates

  • Create promotional pricing for specific date ranges
  • Override base rates during special periods
  • Perfect for holidays, seasons, or last-minute deals
  • Multiple specials can run simultaneously

API Endpoints

Get Base Rates (Public)

GET /api/admin/rates/base

Calculate Booking Cost (Public)

POST /api/admin/rates/calculate
{
  "checkin_date": "2025-12-01",
  "checkout_date": "2025-12-05",
  "guests": 2
}

Manage Rates (Admin)

POST   /api/admin/rates/        # Create rate
PUT    /api/admin/rates/{id}   # Update rate
DELETE /api/admin/rates/{id}   # Deactivate rate
GET    /api/admin/rates/       # List all rates

Database Schema

CREATE TABLE rates (
  id SERIAL PRIMARY KEY,
  rate_type VARCHAR(50) NOT NULL,  -- 'base' or 'special'
  guests INTEGER NOT NULL,          -- 1 or 2
  amount NUMERIC(10, 2) NOT NULL,
  start_date DATE,                  -- NULL for base rates
  end_date DATE,                    -- NULL for base rates
  description VARCHAR(255),
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);

Creating Special Offers

  1. Login to admin dashboard
  2. Navigate to Rates tab
  3. Click "Add Rate" ‚Üí Select "Special Rate"
  4. Fill in details:
    • Guests: 1 or 2
    • Amount: e.g., R650
    • Date range: Start and end dates
    • Description: "Start of holidays special"
  5. Save - appears on homepage immediately!

üìÖ Calendar Synchronization (iCal)

Synchronize bookings across multiple platforms to prevent double-bookings.

Supported Platforms

  • ‚úÖ Airbnb
  • ‚úÖ Booking.com
  • ‚úÖ VRBO / HomeAway
  • ‚úÖ LekkeSlaap
  • ‚úÖ Google Calendar
  • ‚úÖ Any iCal-compatible platform

Export Bookings

Share your bookings with other platforms:

GET /api/ical/bookings.ics

Import this URL into Airbnb, Booking.com, etc. to block dates.

Import External Bookings

Import bookings from other platforms:

POST /api/ical/import
{
  "url": "https://airbnb.com/calendar/ical/xyz123.ics",
  "source": "airbnb"
}

Quick Setup

  1. Get your export URL: https://yoursite.com/api/ical/bookings.ics
  2. Add it to each platform's calendar import settings
  3. Add each platform's export URL to your admin panel
  4. Bookings sync automatically!

Full details: iCal Sync Documentation
